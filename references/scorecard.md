# Skill Quality Scorecard

Use this scorecard when reviewing a candidate skill.

## Score each area from 0 to 2

- **Trigger clarity**
  - 0 = vague
  - 1 = partially clear
  - 2 = explicit use cases and boundaries

- **Non-trigger clarity**
  - 0 = missing
  - 1 = weak
  - 2 = clearly says when not to use

- **Gate strength**
  - 0 = no hard checks
  - 1 = soft suggestions only
  - 2 = explicit must-do checks / blockers

- **Red flags**
  - 0 = none
  - 1 = generic warnings
  - 2 = specific failure signals

- **Fallback path**
  - 0 = none
  - 1 = vague fallback
  - 2 = clear stop/switch/escalation path

- **Output shape**
  - 0 = unconstrained
  - 1 = somewhat structured
  - 2 = explicit output format

- **Cost efficiency**
  - 0 = likely to waste context or time
  - 1 = neutral
  - 2 = clearly cheaper than improvising

- **Trust surface**
  - 0 = opaque / high blast radius
  - 1 = mixed
  - 2 = inspectable / proportionate / low dependency

## Quick interpretation

- **13–16** → strong skill
- **9–12** → usable, but review carefully
- **5–8** → weak / situational
- **0–4** → reject
